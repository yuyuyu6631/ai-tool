# 05 接口规范

## 1. 当前实现说明

当前前端通过 `NEXT_PUBLIC_API_BASE_URL` 请求 FastAPI 服务，默认地址为 `http://localhost:8000`。  
当前公开站点、评论链路、后台管理和认证能力都已基于真实后端路由。

## 2. 当前 API 分类

### 2.1 目录与展示

- `GET /api/tools`
- `GET /api/tools/{slug}`
- `GET /api/tools/search-index`
- `GET /api/tools/import-preview/validation`
- `GET /api/categories`
- `GET /api/categories/{slug}/tools`
- `GET /api/home`
- `GET /api/rankings`
- `GET /api/rankings/{slug}`
- `GET /api/scenarios`
- `GET /api/scenarios/{slug}`

`GET /api/tools` 当前支持：

- `q`
- `category`
- `tag`
- `status`
- `price`
- `access`
- `price_range`
- `sort`
- `view`
- `page`
- `page_size`

`GET /api/tools` 返回 `ToolsDirectoryResponse`，当前包含：

- `items`：工具列表
- `total`、`page`、`pageSize`、`hasMore`
- `categories`、`tags`、`statuses`：分面筛选项
- `priceFacets`、`accessFacets`、`priceRangeFacets`：扩展筛选项
- `presets`：预设视图列表

`GET /api/categories` 当前支持：

- `include_empty`（默认 false，控制是否返回无工具的空分类）

### 2.2 评论

- `GET /api/tools/{tool_slug}/reviews`
- `GET /api/tools/{tool_slug}/reviews/me`
- `PUT /api/tools/{tool_slug}/reviews/me`

说明：

- 公开评论接口返回评分汇总、编辑评论和用户评论
- `/me` 相关接口依赖当前登录用户

### 2.3 认证

- `POST /api/auth/register`
- `POST /api/auth/login`
- `POST /api/auth/logout`
- `GET /api/auth/me`

当前用户响应体至少包含：

- `id`
- `username`
- `email`
- `status`
- `role`
- `createdAt`

### 2.4 AI 与推荐

- `GET /api/ai-search`
- `POST /api/recommend`
- `POST /api/chat`

说明：

- `GET /api/ai-search` 返回 `AiSearchResponse`
- `POST /api/recommend` 返回 `RecommendItem[]`
- `POST /api/chat` 为 SSE 流式接口

### 2.5 抓取与解析

- `POST /api/crawl/jobs`
- `POST /api/parser/extract`

当前状态：

- 抓取与解析入口已存在
- 但尚未形成完整审核发布闭环

### 2.6 后台管理接口

这些接口要求管理员身份：

- `GET /api/admin/overview`
- `GET /api/admin/tools`
- `POST /api/admin/tools`
- `GET /api/admin/tools/{tool_id}`
- `PUT /api/admin/tools/{tool_id}`
- `GET /api/admin/reviews`
- `DELETE /api/admin/reviews/{review_id}`
- `GET /api/admin/rankings`
- `POST /api/admin/rankings`
- `GET /api/admin/rankings/{ranking_id}`
- `PUT /api/admin/rankings/{ranking_id}`

## 3. 典型结构说明

### 3.1 AI 搜索

`GET /api/ai-search` 返回：

- `mode`
- `query`
- `normalized_query`
- `ai_panel`
- `results`
- `directory`
- `meta`

### 3.2 评论接口

`GET /api/tools/{tool_slug}/reviews` 返回：

- `summary`
- `editorReviews`
- `userReviews`

`PUT /api/tools/{tool_slug}/reviews/me` 请求体：

```json
{
  "rating": 4,
  "title": "适合中文办公场景",
  "body": "总结速度快，但复杂格式还需要手动调整。"
}
```

### 3.3 后台总览

`GET /api/admin/overview` 返回：

- `toolCount`
- `draftToolCount`
- `publishedToolCount`
- `reviewCount`
- `rankingCount`
- `recentUpdatedTools`

## 4. 异常响应

当前后端主要使用 FastAPI 标准错误结构：

```json
{
  "detail": "错误说明"
}
```

典型状态码：

- 参数错误：`422`
- 未登录或会话失效：`401`
- 无管理员权限：`403`
- 资源不存在：`404`
- 冲突：`409`
- 服务端异常：`500`
