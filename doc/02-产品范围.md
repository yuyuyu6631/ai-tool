# 产品范围

## 当前已实现范围

### 公开页面与承接入口

当前已落地并可访问的主要页面：

1. 首页 `/`
2. 工具详情 `/tools/[slug]`
3. 场景列表 `/scenarios`（当前由首页侧栏场景组件承接，前端无独立路由页面；API `/api/scenarios` 独立可用）
4. 场景详情 `/scenarios/[slug]`（同上，由首页侧栏承接；API `/api/scenarios/{slug}` 独立可用）
5. 对比页 `/compare/[comparisonSlug]`
6. 匹配页 `/matches`
7. 认证页 `/auth`

当前兼容入口：

- `/tools`：保留旧入口，但永久重定向到 `/`
- `/rankings`：当前直接重定向到 `/`

### 公开能力

当前已经在代码中落地的能力：

1. 首页统一搜索/筛选承接
2. 首页按专家推荐优先及内容分级策略（Head/Mainstream 等）渲染
3. 视图切换（推荐 / 热门 / 最新 / 免费 / 国产）
4. 目录筛选：搜索、分类、标签、价格、访问限制、排序、分页
4. 工具详情：基础信息、评分、评论摘要、相似工具、场景推荐
5. 用户认证：注册、登录、登出、恢复登录态
6. 用户评论：查看已发布评论、查看自己的评论、提交/更新自己的评论
7. 对比页：2 到 3 个工具横向比较
8. 后台管理：总览、工具管理、评论管理、榜单管理（已集成细粒度的 RBAC 角色操作权限保护）
9. AI 搜索：`GET /api/ai-search`
10. RAG 对话：`POST /api/chat`
11. 数据导入/校验与轻量抓取入口

## 当前不在已完成范围内的内容

以下内容当前仍未形成完整闭环，或仍停留在规划阶段：

- 收藏历史与个人中心
- 支付、订单、会员体系
- 第三方 OAuth 社交登录
- 完整抓取审核发布闭环
- SEO 专题生产系统
- 多轮聊天会话管理与历史归档
- 更完整的用户社区互动体系

```text
首页 /
├── 搜索与筛选承接
├── 分类入口
├── 视图切换（推荐 / 热门 / 最新 / 免费 / 国产）
├── 工具卡片列表
├── 场景入口
└── 热门榜单区

工具详情 /tools/[slug]
├── 基础信息
├── 评分与评论摘要
├── 编辑点评
├── 场景推荐
└── 相似工具

场景 /scenarios
└── 场景列表

场景详情 /scenarios/[slug]
└── 场景说明 + 主推/备选工具

对比 /compare/[comparisonSlug]

├── 登录

└── 已登录状态展示

后台 /admin
├── 总览
├── 工具管理
├── 评论管理
└── 榜单管理
```

## 当前边界说明

- 当前项目已经不是“只有静态演示”的纯前端 Demo
- 当前项目也还不是完整商业化产品，仍处于 MVP 演进阶段
- 后台与评论已经进入当前事实，但抓取审核和商业化仍未闭环
- 文档中的未来能力仅作规划，不应误认为已经在代码中落地
