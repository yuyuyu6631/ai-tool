### 4.3 已采用色彩方案

当前前端实际采用的是一套“天空渐层背景 + 半透明玻璃卡片 + 深色主操作”的轻玻璃工作台配色，不是旧版暖灰产品风。

#### 基础主色板

| 角色 | 色值 | 当前用途 |
| --- | --- | --- |
| 页面主背景 | `#eef4fb` | 页面底色、天空感氛围基底 |
| 主文字 | `#0f172a` | 标题、主信息 |
| 玻璃卡片底 | `rgba(255, 255, 255, 0.5)` | 面板、卡片基础底 |
| 次级玻璃底 | `rgba(255, 255, 255, 0.2)` | 次级按钮、弱卡片底 |
| muted 背景 | `rgba(255, 255, 255, 0.22)` | 标签、弱态区块、图标底板 |
| 输入框底 | `rgba(255, 255, 255, 0.7)` | 搜索输入区 |
| 次级文字 | `#4f5f78` / `#42536b` | 说明文案、辅助信息 |
| 深色 hover | `#0f172a` / `rgba(15, 23, 42, 0.95)` | 主按钮 hover |
| 强调辅助文字 | `#475569` | 标签与次级强调文字 |
| 评分底色 | `rgba(255, 247, 237, 0.68)` | 分数徽标背景 |
| 评分文字 | `#8a4b10` | 分数徽标文字 |
| 错误色 | `#d4183d` | destructive token |

#### 边框与透明度体系

当前项目大量使用半透明白边框和模糊来建立层次，而不是靠重实色卡片：

- 主边框常用：`rgba(255, 255, 255, 0.24 ~ 0.28)`
- 弱边框常用：`rgba(255, 255, 255, 0.18 ~ 0.2)`
- Hover 边框：`rgba(255, 255, 255, 0.3 ~ 0.36)`
- Header 毛玻璃底：`rgba(255, 255, 255, 0.14)`
- SearchPanel 半透明底：`rgba(255, 255, 255, 0.2)`
- 玻璃模糊常用：`blur(18px ~ 30px) + saturate(145% ~ 150%)`

## 9. 当前资产结构判断

当前项目已经具备以下前端资产基础：

- 一套可运行的 React 单页应用骨架
- 一套稳定的页面和业务组件

- 一组可复用 UI 基础组件

- 一批测试用例

尚未完全打通的部分：

- Logo 与 slug 的映射

- mock 数据到真实知识库的迁移
- 首页风格与榜单、详情页之间的视觉衔接
- 页面级测试覆盖仍偏少
