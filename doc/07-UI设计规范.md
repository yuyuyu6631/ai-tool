# UI / UX 设计规范

## 总体视觉方向
整体风格必须符合以下关键词：

- 简洁
- 现代
- 克制
- 稳定
- 平台感
- 轻内容产品风格

**不要**做成传统企业官网，不要过度炫技，也不要出现明显“AI 模板页”气质。

## 推荐参考风格
设计可参考以下方向，但不要直接照搬：

- Linear / Vercel 式克制感
- Notion 式整洁排版
- Stripe / Apple 官网的节奏控制
- 内容产品常见的信息优先级表达

## 页面表现要求
- 首屏要有冲击力
- 首屏不能过满
- 卡片不能太密、太碎、太平均
- 标题层级清楚
- 标签、评分、按钮视觉统一
- 主色控制在 2 到 3 个以内
- 适合领导快速浏览
- 看上去像"已经有一定基础的平台"

额外要求：

- 不依赖渐变、发光、玻璃拟态制造高级感
- 通过留白、比例、字重、对齐体现审美
- 每个区块都要有明确主次，不能同等强调

## 配色风格建议
建议使用浅色中性底 + 深色主操作按钮的方案。

重点不是极致设计，而是让页面具备：
- 可信度
- 产品感
- AI 行业调性
