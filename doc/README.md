# AI 工具评测平台文档索引

本目录用于沉淀项目需求、设计规范、技术方案与前端落地资产。

## 当前状态

- 原始方案文档中曾以 `Next.js + Tailwind CSS` 作为推荐实现。
- 当前实际落地前端为 `React 18 + Vite 6 + React Router 7 + Tailwind CSS 4`。
- 站点已实现首页、榜单页、工具详情页、场景列表页、场景详情页，并使用本地 `mock API + JSON 数据` 完成演示闭环。
- 本次已补充“知识库升级”与“前端实际已采用资产文档”，用于统一团队认知，避免继续按旧栈沟通。
